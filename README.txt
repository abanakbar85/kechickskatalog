Kemchicks Katalog Project (React + Vite + Supabase)

Cara Deploy ke Vercel:
1. Buka https://vercel.com
2. Klik New Project → Import Project → Upload file zip ini
3. Pilih framework: Vite
4. Tambahkan environment variables:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
5. Klik Deploy

Website akan aktif di: https://<nama-project>.vercel.app
